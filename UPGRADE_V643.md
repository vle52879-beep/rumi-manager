# RUMI Manager v6.4.3 — Multi Admin

## Tính năng mới

- Admin hiện tại có thể tạo thêm tài khoản quản trị tại **Tài khoản & bảo mật**.
- Mỗi admin dùng tên đăng nhập và mật khẩu riêng, có đầy đủ quyền quản trị.
- Admin mới bắt buộc đổi mật khẩu trong lần đăng nhập đầu tiên.
- Danh sách admin hiển thị trạng thái, lần đăng nhập cuối và tài khoản đang sử dụng.
- Hoạt động tạo admin được ghi vào nhật ký bảo mật.

## Cập nhật

Không cần chạy SQL mới. Deploy source, đăng nhập admin chính, mở **Tài khoản & bảo mật** rồi điền biểu mẫu **Tạo admin mới**.
